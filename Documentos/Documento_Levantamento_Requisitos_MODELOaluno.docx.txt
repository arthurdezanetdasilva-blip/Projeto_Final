1.2 Escopo
COMO PREENCHER
Descreva o que o sistema vai abranger (os módulos e funções principais que serão atendidos). Se for útil, indique também o que fica de fora desta versão.
	O Sistema de Gestão Escolar será responsável por auxiliar a coordenação de uma instituição de ensino no gerenciamento de professores, semestres, disciplinas, feriados e cronogramas de aulas.
O sistema permitirá o cadastro dessas informações, o cálculo automático dos custos das disciplinas e a emissão de relatórios administrativos.
Nesta versão não serão desenvolvidos aplicativo para dispositivos móveis nem integração com outros sistemas.

2. Descrição geral do sistema
COMO PREENCHER
Apresente o sistema em um ou dois parágrafos, em visão de alto nível, antes dos requisitos detalhados: o que ele faz, para quem, qual problema resolve e quais são os principais módulos.
	

O Sistema de Gestão Escolar será desenvolvido para facilitar a administração acadêmica de uma instituição de ensino.
O sistema permitirá cadastrar professores, semestres, disciplinas e feriados, gerar automaticamente cronogramas de aulas, calcular o custo das disciplinas com base na carga horária e no valor da hora-aula do professor, além de emitir relatórios administrativos.
O sistema será desenvolvido para ambiente Web, podendo ser acessado por navegadores modernos.

8. Restrições e premissas
COMO PREENCHER
Restrições: limitações conhecidas que o projeto precisa respeitar (prazo, tecnologia, ambiente de execução).
Premissas: suposições assumidas pela equipe que, se mudarem, afetam o projeto.
	



Restrições:
O sistema será desenvolvido para ambiente Web.
Será utilizado banco de dados MySQL ou LocalStorage para armazenamento das informações.
Apenas usuários cadastrados poderão acessar o sistema.
O desenvolvimento deverá respeitar o cronograma da disciplina.
Premissas:
O professor fornecerá todas as informações necessárias para o desenvolvimento.
Os usuários terão acesso à internet.
Os dados cadastrados serão verdadeiros.
O sistema será utilizado apenas pela instituição de ensino.
